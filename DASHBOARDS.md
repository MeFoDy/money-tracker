# Требования к экрану финансовых дашбордов

## 🎯 Концепция экрана

**Принцип**: три уровня восприятия — *взгляд (2 сек)* → *анализ (30 сек)* → *детализация (по запросу)*

---

## 📊 Блок 1 — KPI-плашки (верхняя строка)

Быстрые метрики без графиков, читаются за 2 секунды.

| Метрика | Описание |
|---|---|
| Итоговый баланс | доходы − расходы за период |
| Суммарные доходы | с дельтой к предыдущему периоду |
| Суммарные расходы | с дельтой к предыдущему периоду |
| Топ-категория расходов | название + сумма |
| Количество транзакций | общее за период |

**Требования:**
- Цветовая индикация: зелёный / красный / нейтральный
- Дельта отображается стрелкой + процентом
- Клик на плашку → фильтрует все графики ниже

---

## 📈 Блок 2 — Доходы и расходы во времени

**График:** Line Chart или Bar Chart (переключатель)

```
Ось X: даты (день / неделя / месяц — группировка)
Ось Y: сумма в валюте
Линии: отдельно доходы (зелёная) и расходы (красная)
Третья линия: накопленный баланс (пунктир, синяя)
```

**Требования:**
- Тултип при hover: дата, сумма дохода, сумма расхода, баланс на момент
- Переключатель группировки: `день / неделя / месяц`
- Зона между линиями закрашена (зелёная если доход > расход, красная — наоборот)
- Клик на точку → модальное окно со списком транзакций за этот период

---

## 🍩 Блок 3 — Структура расходов по категориям

**График:** Doughnut Chart (основной) + горизонтальный Bar Chart (опциональный вид)

**Требования:**
- Цвета сегментов берутся из парсера (степень полезности)
- В центре дoнатa: общая сумма расходов
- Легенда справа: категория + сумма + % от общего
- Легенда сортируется по убыванию суммы
- Hover на сегмент → подсвечивается + показывает тултип
- Клик на сегмент → drill-down: тот же дoнат, но показывает подкатегории или отдельные транзакции

---

## 🔥 Блок 4 — Тепловая карта расходов по дням

**График:** кастомная сетка через Chart.js или matrix-plugin

```
Ось X: день недели (Пн–Вс)
Ось Y: неделя периода
Ячейка: интенсивность цвета = сумма расходов
```

**Требования:**
- Показывает паттерны трат (например, «по пятницам трачу больше»)
- Hover → тултип с суммой и датой
- Клик → список транзакций за день
- Переключатель: расходы / доходы / количество транзакций

---

## 📉 Блок 5 — Топ-категории: план vs факт / сравнение периодов

**График:** Grouped Bar Chart

```
По оси X: категории
По оси Y: суммы
Группы: текущий период vs предыдущий период
```

**Требования:**
- Сортировка по наибольшему отклонению (сразу видно, где перерасход)
- Цвет баров — из парсера (полезность категории)
- Тултип: разница в рублях и процентах
- Опционально: горизонтальная линия «лимит» если пользователь задал бюджет

---

## 🔍 Блок 6 — Детальная таблица транзакций (drill-down)

Появляется при клике на любой элемент графиков выше.

**Требования:**
- Открывается как боковая панель или модальное окно (не уходить со страницы)
- Колонки: дата, описание, категория, сумма, счёт, цвет полезности
- Сортировка по каждой колонке
- Поиск по описанию
- Кнопка «Экспорт CSV» для выбранного среза

---

## 🎛️ Глобальные фильтры (закреплены вверху страницы)

```
[ Период: от—до ]  [ Счёт: все / №счёта ]  [ Категории: мультиселект ]  [ Тип: доход/расход/всё ]
```

**Требования:**
- Применяются ко всем блокам одновременно
- При изменении фильтра — анимированное обновление графиков (Chart.js update)
- Состояние фильтров сохраняется в URL (shareable ссылка)
- Кнопка «Сбросить фильтры»

---

## ⚙️ Технические требования Chart.js

```javascript
// Обязательные плагины
- chartjs-plugin-datalabels   // подписи на графиках
- chartjs-plugin-zoom         // zoom/pan для временных рядов
- chartjs-plugin-annotation   // линии лимитов/целей

// Поведение
- Все анимации: duration 400ms, easing 'easeInOutQuart'
- Responsive: true, maintainAspectRatio: false
- Единый theme-объект с цветами (light/dark mode)
- Тултипы кастомные: HTML-тултипы через external tooltip callback
```

---

## 📱 Адаптивность и UX

| Breakpoint | Раскладка |
|---|---|
| Desktop (>1200px) | 2 колонки: KPI + линейный, донат + тепловая карта |
| Tablet (768–1200px) | 1 колонка, графики полной ширины |
| Mobile (<768px) | только KPI-плашки + донат, остальное под аккордеоном |

**Доп. требования:**
- Skeleton-loader пока данные парсятся
- Empty state если данных за период нет
- Все числа форматируются через `Intl.NumberFormat` (разряды, валюта)
- Доступность: aria-labels на canvas элементах
